# Mobile App

Built with **Expo SDK 54** and React Native.

## Tech Stack

- **Framework**: Expo SDK 54
- **Navigation**: Expo Router
- **State**: Zustand
- **Storage**: MMKV + SQLite
- **UI**: Custom components with Lucide icons

## Features

- Smart Notes with Markdown/HTML editor
- Mood Diary with emotional tracking
- AI auto-tagging and summaries
- Offline support with background sync
