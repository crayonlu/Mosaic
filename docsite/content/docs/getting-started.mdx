# Getting Started

## Prerequisites

- [Docker](https://docker.com) & Docker Compose
- A server with a public IP

## Step 1: Deploy the Server

Create a `.env` file and `docker-compose.yml`, then start:

```bash
docker-compose up -d
```

For detailed configuration, see the [Server Deployment](/docs/server/deployment) guide.

## Step 2: Connect the Mobile App

Download the latest APK from the [Releases page](https://github.com/crayonlu/Mosaic/releases), install on your Android device, and enter your server URL.
