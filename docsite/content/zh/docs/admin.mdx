# 管理面板

Mosaic 包含一个基于 Web 的管理面板，用于管理服务端、AI 配置和自动化设置。

## 访问

在浏览器中打开 `http://你的服务器地址:8080/admin`。

**默认凭据**：
- 用户名：`admin`
- 密码：`admin123`

生产环境中通过 `ADMIN_USERNAME` 和 `ADMIN_PASSWORD` 环境变量修改。

## 仪表盘

主仪表盘展示：

- **服务端健康**：运行时间、存储用量、数据库大小
- **统计**：笔记、日记、资源和机器人的总数
- **最近活动**：近期服务端事件日志

## AI 配置

配置服务端使用的 AI 提供者：

- **Bot 配置**：用于对话生成、标签、摘要和日记生成
- **嵌入配置**：用于生成向量嵌入以支持语义搜索

每个配置支持：
- 提供者选择：**OpenAI** 或 **Anthropic**
- Base URL、API Key 和模型名称
- Temperature 和 max tokens
- 视觉和推理能力开关（自动检测）

详细设置参见 [AI 功能](/zh/docs/ai-features)。

## 自动化

控制哪些 AI 功能自动运行：

| 设置项 | 默认 | 说明 |
|--------|------|------|
| 自动标签 | 开启 | 创建笔记时自动打标签 |
| 自动摘要 | 关闭 | 为每条笔记生成 AI 摘要 |
| 自动日记 | 开启 | 自动生成每日日记 |
| 最少笔记数 | 2 | 触发自动日记的最低笔记数 |
| 最少字符数 | 150 | 触发自动日记的最低总字符数 |
| 时区 | Asia/Shanghai | 日记日期计算的时区 |

## 机器人管理

在机器人页面创建、编辑和删除 AI 机器人。每个机器人可以设置：

- 名称和头像
- 描述和标签
- 自动回复开关（回复匹配的笔记）
- 排序顺序

## 记忆系统

记忆面板显示向量嵌入的状态。你可以触发**回填**来为尚未生成嵌入的笔记生成向量嵌入，这是语义搜索和机器人记忆功能所必需的。
