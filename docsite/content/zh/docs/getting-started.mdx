# 快速开始

## 前提条件

- [Docker](https://docker.com) 和 Docker Compose
- 一台拥有公网 IP 的服务器

## 第一步：部署服务端

创建 `.env` 文件和 `docker-compose.yml`，然后启动：

```bash
docker-compose up -d
```

## 第二步：连接移动端

从 [Release 页面](https://github.com/crayonlu/Mosaic/releases) 下载最新 APK，安装到 Android 设备上，输入你的服务器地址即可。
