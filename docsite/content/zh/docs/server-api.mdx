# API 参考

> 版本：v2.6 | 更新日期：2026-05-16

## 基础信息

Base URL：`http://localhost:8080`

认证方式：Bearer Token

## 认证

### POST /api/auth/login

```json
{ "username": "admin", "password": "admin123" }
```

响应：

```json
{
  "accessToken": "eyJ...",
  "refreshToken": "eyJ...",
  "user": { "id": "uuid", "username": "admin" }
}
```

### POST /api/auth/refresh

```json
{ "refreshToken": "eyJ..." }
```

## Memo

### GET /api/memos

分页查询。Query：`page`、`pageSize`、`archived`、`diaryDate`、`search`

### POST /api/memos

```json
{ "content": "# 笔记内容", "tags": ["工作"], "diaryDate": "2026-05-16" }
```

### GET /api/memos/:id

### PUT /api/memos/:id

### DELETE /api/memos/:id
