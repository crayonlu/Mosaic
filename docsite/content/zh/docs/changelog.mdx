# 更新日志

## v1.0.1 (2026-05-22)

### 🎨 新主题系统

- 移除了旧的浅色/深色切换，新增**主题选择器** — 可选 **Quiet Paper** 和 **Clean Slate** 两种主题
- 新用户首次启动时选择主题
- 主题偏好会持久化保存

### 📖 全新文档站

- 完全重新设计的首页和文档布局
- **中英双语支持**，一键切换语言
- 全站全文搜索
- 新增 18 篇文档，涵盖：环境配置、FAQ、服务端部署、故障排查、AI 配置、同步、管理后台、隐私政策、数据库等

### 📱 移动端改进

- 搜索输入增加防抖处理，体验更流畅
- AI 摘要改为 Markdown 渲染（之前为纯文本）
- 情绪背景和 Bot 回复增加了动画过渡效果
- Memo 卡片界面精简，减少操作按钮
- **恢复 Expo Go 兼容性**（延迟加载原生模块）
- 图片加载可靠性提升

### 🔒 安全与可靠性

- CORS 策略可通过 `ALLOWED_ORIGINS` 环境变量配置白名单
- 上传文件大小限制（资源文件 100MB，头像 10MB）
- API 错误响应不再泄露内部错误细节
- 本地文件存储增加路径穿越防护
- 资源删除操作增加用户归属校验
- `ADMIN_PASSWORD` 现在为必填项（不再有默认密码）
- docker-compose 不再暴露 PostgreSQL 端口

### ⚡ 性能优化

- 优化备忘录列表加载 — 消除 N+1 资源查询问题（服务端 + 离线模式）
- 同步服务的日记查询改为批量获取
- 延迟初始化代码高亮（lowlight）和 Markdown 解析器
- Docker 构建层缓存优化

### 🛠 其他

- CI 流程精简（代码检查 + 类型检查 + 构建 → 合并为单一 check 命令）
- 新增 `scripts/fmt.sh`（macOS 格式检查脚本）
- Admin UI 主题改进
