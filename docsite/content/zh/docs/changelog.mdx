# 更新日志


## v1.0.4 (2026-06-13)

### 🖥 Dashboard

- **Bot 配置管理。** Bot 编辑功能已从移动端完全迁移至 Dashboard（`/bots`）。名称、描述、标签、自动回复、头像、模型 — 所有字段集中管理。
- **模型覆盖字段。** Bot 编辑器新增 `model` 字段，支持按 Bot 覆盖默认模型（留空则使用系统默认）。
- **活动刷新优化。** 点击刷新不再闪现骨架屏占位块，旧数据保持可见直到新数据加载完成。
- **主题色彩修复。** Clean Slate 主题下 accent 前景色从近黑色改为白色，在绿色背景上清晰可读。

### 📱 移动端

- **Bot 列表改为只读。** 设置中移除 Bot 编辑功能，保留名称、头像、标签展示，自动回复状态以胶囊徽章呈现（Auto 实心 / Manual 描边）。

## v1.0.3 (2026-06-08)

### 🤖 AI 日记归档

- **Memo 图片现在会随文本一起发送给 AI。** 每日日记归档现在会将 Memo 图片与文本一同发送。每条 Memo 独立为一条消息，配图内联在对应消息中，AI 能正确关联图片与文本上下文。限制：每条 Memo 最多 2 张图片，每次归档最多 6 张。
- **超限图片以文字描述提供。** 超出限制的图片会通过 AI 生成文字描述，作为文本上下文附加到对应 Memo，确保没有图片信息被完全丢弃。
- **持久化图片描述。** 图片上传时即生成 AI 描述并存入 `resources.ai_description` 列。日记归档复用这些已存储的描述，避免重复调用视觉模型。

### 📱 显示

- **高清图片开关。** 设置 → 通用。开启后 Memo 卡片网格和全屏预览直接展示原图，无需缩略图→查看原图的步骤。关闭时保留原有的模糊到清晰渐进加载流程。设置持久化到设备存储。

### 🤖 Bot 编辑器

- FPS 性能监控
- 描述编辑器

### 🐛 修复

- 首页和 Bot 编辑器的键盘处理优化
- SafeKeyboardProvider 边界情况修复
- 键盘切换和折叠状态下的掉帧修复

 ## v1.0.2 (2026-05-28)

### ⚡ 性能优化

- **修复键盘卡顿** — BotEditorSheet 改用 `KeyboardAwareScrollView`（UI 线程原生动画），消除软键盘弹出/收起时的掉帧卡顿
- 升级 `react-native-keyboard-controller` 1.18.5 → 1.21.8 — 带来 layout-free 键盘动画、内存泄漏修复、Fabric 兼容性提升
- FullScreenEditor 从 React Native 内置 `KeyboardAvoidingView` 迁移至 keyboard-controller 原生版本

### 🐛 问题修复

- 修复归档弹窗关闭后切换日期会意外重新弹出的问题（BottomSheetModal props 变化导致 re-present）
- 修复归档选择模式下日历选择器仍可点击的问题 — 现在进入选择模式后正确禁用
- 修复日期筛选器的清除（X）按钮在归档选择期间仍可点击
- 修复 Button 组件的 `style` prop 无法影响布局的问题（如 footer 按钮的 `flex: 1` 不生效）— style 现在正确应用于外层 Pressable

### 🔧 重构

- 提取 `SettingsSection` 可复用组件 — 所有设置项统一使用折叠模式（图标 + 标题 + 摘要 + 徽章 + 可折叠内容）
- 关于页面重新设计，与其他折叠 section 保持一致，展示版本信息、更新状态和操作按钮
- 新增 `useSafeKeyboardHandler` hook 确保 Expo Go 兼容性

### 🌐 国际化

- 新增 i18n key：`version`、`updateStatus`、`installUpdate`、`checkUpdate`（中英文）

### 📱 移动端其他改进

- 应用内更新系统，通过 GitHub Releases 下载（含下载进度、安装提示）
- 全新 App 图标和启动屏，品牌视觉焕新
- BootSplash 集成原生启动屏
- 术语统一："助手" → "Bot"
- 完整迁移至 `react-i18next` 国际化方案
- 日记功能增强：新增 hooks 和翻页动画
- 异步操作增加指数退避重试机制
- Android 发布构建增加 ProGuard 规则插件

### 🖥 服务端

- 修复 embedding 执行顺序 — 现在在 Bot 回复前执行，确保 RAG 锚点 embedding 存在
- AI 服务调用中 `max_tokens` 改为从配置读取（不再硬编码）
- resources 表新增 `user_id` 字段用于归属追踪

## v1.0.1 (2026-05-22)

### 🎨 新主题系统

- 移除了旧的浅色/深色切换，新增**主题选择器** — 可选 **Quiet Paper** 和 **Clean Slate** 两种主题
- 新用户首次启动时选择主题
- 主题偏好会持久化保存

### 📖 全新文档站

- 完全重新设计的首页和文档布局
- **中英双语支持**，一键切换语言
- 全站全文搜索
- 新增 18 篇文档，涵盖：环境配置、FAQ、服务端部署、故障排查、AI 配置、同步、管理后台、隐私政策、数据库等

### 📱 移动端改进

- 搜索输入增加防抖处理，体验更流畅
- AI 摘要改为 Markdown 渲染（之前为纯文本）
- 情绪背景和 Bot 回复增加了动画过渡效果
- Memo 卡片界面精简，减少操作按钮
- **恢复 Expo Go 兼容性**（延迟加载原生模块）
- 图片加载可靠性提升

### 🔒 安全与可靠性

- CORS 策略可通过 `ALLOWED_ORIGINS` 环境变量配置白名单
- 上传文件大小限制（资源文件 100MB，头像 10MB）
- API 错误响应不再泄露内部错误细节
- 本地文件存储增加路径穿越防护
- 资源删除操作增加用户归属校验
- `ADMIN_PASSWORD` 现在为必填项（不再有默认密码）
- docker-compose 不再暴露 PostgreSQL 端口

### ⚡ 性能优化

- 优化备忘录列表加载 — 消除 N+1 资源查询问题（服务端 + 离线模式）
- 同步服务的日记查询改为批量获取
- 延迟初始化代码高亮（lowlight）和 Markdown 解析器
- Docker 构建层缓存优化

### 🛠 其他

- CI 流程精简（代码检查 + 类型检查 + 构建 → 合并为单一 check 命令）
- 新增 `scripts/fmt.sh`（macOS 格式检查脚本）
- Admin UI 主题改进
